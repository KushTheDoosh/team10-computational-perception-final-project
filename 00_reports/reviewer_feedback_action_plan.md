# Team 10 Reviewer Feedback -> Final Writeup Actions

## Top recurring feedback themes
1. Scope may be too broad (farm + warehouse + wildlife in one semester).
2. "Deep Learning" title may not match mostly classical methods.
3. Main contribution was not obvious early enough (behavior layer vs base detection).
4. Related work depth likely needs expansion.
5. Some wording was informal or verbose.
6. More detail needed on tracking consistency, threshold selection, and annotation process.

## Concrete actions in final report
1. Narrow declared scope to 1-2 deployment scenarios and explicitly list out-of-scope items.
2. Align title with actual implemented methods.
3. State contributions in the introduction in a numbered list.
4. Add a dedicated subsection defining behavior labels with threshold formulas/rules.
5. Add annotation and ground-truth procedure in Experimental Setup.
6. Include per-condition metrics table plus confusion matrix.
7. Add failure case analysis (low-light, occlusion, background motion).
8. Expand references to instructor expectations and include tracking/anomaly papers.

## Submission quality checks
1. Every major claim has either a metric, a figure, or a citation.
2. No filler wording (e.g., "definitely," "beautifully," "massive").
3. Abstract contains exact quantitative outcomes.
4. Conclusion restates results and limitations, not only project intention.
